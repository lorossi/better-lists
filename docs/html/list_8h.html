Detailed Description

Simple linked list library made because I wanted to kill some time. By default, the data stored inside the list is int. define CHARLIST, STRINGLIST, FLOATLIST, DOUBLELIST, respectively to store chars, strings, floats and double. if you want to use some custom data (struct, for example) define CUSTOMTYPE as the type you want to store. Some functions (like printList) won't work.
